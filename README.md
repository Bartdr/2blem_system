# 2BLE M — Barbershop & Movil

Sitio web 2 en 1 para:
- **2BLE M Barbershop** — Reserva de citas, turnos en espera, servicios y galería
- **2BLE M Movil** — Catálogo de teléfonos, especificaciones y reparación

## 📁 Archivos
```
index.html   → Página principal con las dos secciones
style.css    → Estilos
app.js       → Lógica y catálogo de teléfonos
```

## 🚀 Publicar en GitHub Pages

1. Crea un repositorio en GitHub (ej: `2blem-website`)
2. Sube los 3 archivos: `index.html`, `style.css`, `app.js`
3. Ve a **Settings → Pages**
4. En **Source** selecciona `main` y carpeta raíz `/`
5. Guarda — en unos minutos estará en: `https://tuusuario.github.io/2blem-website/`
